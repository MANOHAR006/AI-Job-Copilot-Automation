# AI Job Copilot – Automated Job Discovery

## Overview
AI Job Copilot is an automation-driven system that helps job seekers discover relevant job opportunities faster by reducing manual effort.

## Problem
Job seekers spend hours searching across multiple platforms and tracking opportunities manually.

## Solution
This system automates job discovery, filtering, tracking, and alerts using workflow automation.

## Features
- Automated job search
- AI-based filtering
- Duplicate removal
- Structured tracking in Google Sheets
- Email notifications
- Scalable automation workflow

## Tech Stack
- n8n workflow automation
- Google Sheets
- APIs for job search
- Email automation

## Workflow
1. Fetch user preferences
2. Automated job search
3. AI filtering
4. Store jobs in Google Sheets
5. Send alerts

## Impact
- Saves time
- Improves job discovery
- Reduces manual effort
